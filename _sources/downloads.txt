Stable

`SpikeSort 0.12`_ 

The most recent stable release is also available from `GitHub`_.

Development
-----------

All development is done on GitHub on *dev* branch or feature branches.
If you want to use the most recent code, please clone the `dev`_
branch.

If you want to contribute to the code we will happily accept pull
requests to the *dev* branch.
